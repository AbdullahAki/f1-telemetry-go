## Build & Run

1. First, install repository

```bash
git clone https://github.com/anilmisirlioglu/f1-telemetry-go && cd f1-telemetry-go
```

2. Then, run on docker (or just write `go run .` and run sample app)
```bash
cd docker && docker-compose up -d
```
